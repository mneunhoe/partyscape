# Effective number of parties (Laakso–Taagepera)

`enp(p) = 1 / hhi(p)`. Equal to
[`hill_number()`](https://mneunhoe.github.io/partyscape/reference/hill_number.md)
at q = 2. `effective_parties()` is an alias.

## Usage

``` r
enp(p)

effective_parties(p)
```

## Arguments

- p:

  numeric vector / matrix / data.frame of shares.
