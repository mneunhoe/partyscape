# Plot a diversity_profile

Thin wrapper around
[`autoplot.diversity_profile()`](https://mneunhoe.github.io/partyscape/reference/partyscape-autoplot.md)
that prints the resulting ggplot. Returns the ggplot invisibly for
further composition.

## Usage

``` r
# S3 method for class 'diversity_profile'
plot(x, ...)
```

## Arguments

- x:

  a `diversity_profile` object.

- ...:

  forwarded to
  [`autoplot.diversity_profile()`](https://mneunhoe.github.io/partyscape/reference/partyscape-autoplot.md).
